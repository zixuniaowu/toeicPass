可复用性设计要点

设计决策说明
IAdStore 接口宿主应用只需实现 3 个属性即可接入任何数据库
AdApiFunctions 注入React 组件通过 Props 接收 API 函数，不绑定特定 HTTP 客户端
React optional peerDep只用后端时不需要安装 React
零框架依赖后端纯 TypeScript，NestJS/Express/Fastify/Hono 均可使用
内置种子数据一行代码 seedIfEmpty(DEFAULT_AD_SEEDS) 引导
双入口导出@toeicpass/ad-system (后端) + @toeicpass/ad-system/web (前端)
