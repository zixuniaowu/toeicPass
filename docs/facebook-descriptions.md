🎧 English & Japanese pronunciation practice app ── toeicPass is now open source!

Practice shadowing with TED/YouTube materials, real-time pronunciation analysis & correction with Web Speech API.
SM-2 spaced repetition to memorize pronunciation patterns. Includes 500+ vocabulary & grammar pronunciation exercises.

Browser-based, no installation needed. Built with Next.js + NestJS.

Login with default account.
