# Changelog

All notable changes to `@toeicpass/shared` will be documented in this file.

## [1.0.0] — 2026-04-18

### Added
- `LearningActionCommand` — 6 learning action discriminators
- `LearningPartGroup` — listening/reading discriminator
- `SessionFilters` — practice session filter parameters
- `Locale` — supported UI locales (zh, ja)
- `SessionMode` — diagnostic/practice/mock discriminator
- `Role` — RBAC roles (learner, coach, tenant_admin, super_admin)
- Full specification document (`SPEC.md`)
